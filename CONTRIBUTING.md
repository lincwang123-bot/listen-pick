## Development

```bash
npm install
npm start
npm test
```

## Content Guidelines

All learning content must be suitable for Chinese children ages 6-10.

Use sentences that can be illustrated in five seconds. Avoid abstract ideas, adult work scenes, unsafe behavior, frightening images, culturally narrow references, and exam-style trick questions.

## Pull Request Checklist

- The app still runs locally.
- `npm test` passes.
- New English sentences are grammatically correct.
- Chinese meanings are natural and age-appropriate.
- Correct and wrong images differ by one clear learning point.
- No secrets, local paths, browser profiles, or temporary generated folders are committed.
